# 🚶‍♂️ StepCounter (Pedometer App)

Zamonaviy va minimalistik dizaynga ega bo‘lgan Android pedometr ilova.  
Foydalanuvchining kunlik harakatlarini kuzatadi va maqsadga erishishda yordam beradi.

---

## 📱 Ilova haqida

StepCounter — bu foydalanuvchining kun davomida bosgan qadamlarini, bosib o‘tgan masofasini va yoqilgan kaloriyalarini hisoblaydigan mobil ilova.

Ilova real-time tracking va motivatsion interfeys orqali foydalanuvchini faol hayot tarziga undaydi.

## ⚙️ Asosiy funksiyalar

- ✅ Qadamlarni real-time hisoblash
- 📏 Masofani (km) aniqlash
- 🔥 Kaloriya sarfini hisoblash
- 🎯 Kunlik maqsad qo‘yish (goal tracking)
- 📊 Statistikani ko‘rish
- 🌙 Tema (Dark/Light mode)
- 🔄 Reset (nolga tushirish)
- 💡 Motivatsion xabarlar

---

## 🔁 Ishlash rejimlari

Ilova **2 xil rejimda** ishlaydi:

### 1. 📡 Sensor Mode (Real-time tracking)
- Telefon sensorlari orqali qadamlar aniqlanadi
- Real-time yangilanadi
- Background’da ishlash imkoniyati mavjud

### 2. 📊 Manual / Static Mode
- Oldingi ma’lumotlar asosida statistik ko‘rsatkichlar
- UI preview va demo uchun qulay
- Testing uchun ishlatiladi

## 🧠 Arxitektura

Ilova quyidagi prinsiplar asosida qurilgan:

- **Separation of Concerns**
- **Clean Architecture (qisman)**
- **MVVM pattern**
- **Single Responsibility Principle (SRP)**
